CORE FEATURES (MVP - MUST HAVE)
Reading & Accessibility
1. Text input (paste / type)
2. Text-to-Speech reading
3. Play / Pause / Resume / Stop
4. Adjustable reading speed
5. Line-based text highlighting
6. Dyslexia-friendly font (Lexend / OpenDyslexic)
7. Adjustable font size
8. Adjustable line spacing
9. Background themes (white, yellow, blue, dark)
10. Focused reading UI (minimal distractions)

Usability
11. Save text passages
12. Reopen saved texts
13. Persistent user settings
14. Offline-first functionality

Design Philosophy
15. Dyslexia-first defaults
16. No error-based feedback
17. No time pressure
18. Psychological safety
